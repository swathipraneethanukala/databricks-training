# Day 3 – SQL Joins Assignment

## Topics Covered
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- CROSS JOIN
- Identifying unmatched rows
- Combining data from multiple tables
- Handling NULL values in joins

---

# Concepts Practiced

## 1. LEFT JOIN
Learned how to:
- Display all records from the left table
- Show matching data from the right table
- Identify missing relationships using NULL values

### Used For:
- Finding students without enrollments
- Finding courses without enrollments
- Displaying complete student records

---

## 2. RIGHT JOIN
Learned how to:
- Display all records from the right table
- Retrieve matching data from the left table
- Handle unmatched rows

### Used For:
- Showing all enrollment records with student details

---

## 3. FULL OUTER JOIN
Learned how to:
- Combine all rows from both tables
- Display matched and unmatched records together
- Identify missing relationships on both sides

### Used For:
- Students and enrollments report
- Instructors and courses report

---

## 4. CROSS JOIN
Learned how to:
- Generate all possible combinations between two tables

### Used For:
- Listing every student with every course

---

# Tables Used
- students
- courses
- enrollments
- instructors
