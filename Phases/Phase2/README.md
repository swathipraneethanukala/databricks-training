# SQL to PySpark Phase 2 – Joins & Aggregations Bridge Pack

## 📌 Project Overview
This folder contains the completed assignments for Phase 2 of the Spark pre-training curriculum. The primary goal of this phase is to master **relational joins and aggregations** by bridging the gap between native SQL query design and the native PySpark DataFrame API, working directly with localized schemas inside the Spark Playground environment.

This focuses on:
* Building multi-table relational joins to unify profile and transactional logs.
* Constructing complex aggregations using grouping sets, mathematical sums, averages, and counters.
* Implementing post-aggregation filtering patterns to duplicate relational database mechanics.
